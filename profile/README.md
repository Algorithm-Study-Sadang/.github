### **📌 디렉토리 구조**

       ├ 📦패키지
       ⎮    ├ 📁알고리즘 회차 디렉토리                ex) First, Second.....
       ⎮    ⎮       ├ 📃개인 문제 풀이              ex) Greedy
       ⎮    ⎮       ├ 📃개인 문제 풀이              ex) B1919_Huipyo-Hong
       ⎮    ⎮       ├ 📃개인 문제 풀이              
       ⎮    ⎮       ├ 📝README.md (알고리즘 개념 관련 자료)
       ├ 📝README.md (스터디 관련 설명 README)
